# Linux Shell Basics Demo

This example demonstrates fundamental Linux shell concepts like local and environment variables, their inheritance in child processes, command execution paths, and process IDs. It helps illustrate how the shell manages its environment and executes commands, moving beyond just typing commands.

## Language

`bash`

## How to Run

Save the code as `shell_basics.sh`. Make it executable with `chmod +x shell_basics.sh`. Then run it from your terminal: `./shell_basics.sh`
